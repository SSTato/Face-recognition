# Face-recognition
Face detection and recognition, including face key point detection and correction, using the facenet-retinaface.

### 项目用途
使用facenet-retinaface实现人脸检测与人脸识别，包括人脸关键点检测与矫正，可识别图片、视频、摄像头。
